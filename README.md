# Things Fall Apart Character Frequency Analysis

This project analyzes character appearances in *Things Fall Apart* by Chinua Achebe, counting occurrences of specific characters and visualizing the results.

## Prerequisites

You will need Python installed along with the `matplotlib` library. You can install the required library by running:

```bash
pip install matplotlib
```

## How to Run

1. Make sure `things_fall_apart.txt` is in the same folder as the script.
2. Run the Python script in your terminal:
   ```bash
   python assigment.py
   ```
3. The script will output the frequency of characters in the terminal and generate a `chart.png` file.
4. Open `index.html` in a web browser to view the pie chart on a clean webpage.
